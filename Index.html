Window object and Document object are fundamental components of the Document Object Model(DOM), which represents the structure and content of a web page which allows developers to interact with and manipulate the elements.
The window object represent the browser window or tab that contains the loaded web page. Some key functions of window object includes global scope, window methods, navigation and timers.
On the other hand, the document object represents the web page it self. It allows developers to access and modify the content and structure of the loaded web page. Some key functionalities includes global scope interaction, Accessing the document object and loading and DOMcontentloadedevents.
